# 🇬🇧 French–English Cognates

---

## Why English Helps So Much

سال ۱۰۶۶ میلادی، نورمن‌ها (که فرانسوی حرف می‌زدن) انگلستان رو فتح کردن. نتیجه؟ حدود **۲۸٪ از کل واژگان انگلیسی ریشه‌ی
فرانسوی** داره. حدود ۱۰٬۰۰۰ کلمه‌ی فرانسوی بعد از فتح وارد انگلیسی شد که حدود ۷٬۰۰۰ تاش هنوز زنده‌ست، و حدود **۱٬۷۰۰
تاشون «کاگنیت واقعی»‌ان** — یعنی املا و معنی‌شون تقریباً یکیه.

# Part 1 — Spelling Patterns 🎯

## Pattern 1: `-tion` → `-tion` (almost always identical)

کلمات انگلیسیِ مختوم به **-tion** تقریباً همیشه توی فرانسوی **عیناً همون‌ان**، فقط `[-syohN]` تلفظ می‌شن.

| انگلیسی      | فرانسوی            | تلفظ                   |
|--------------|--------------------|------------------------|
| information  | l'information (f)  | [aN-for-mah-SYOHN]     |
| nation       | la nation          | [nah-SYOHN]            |
| situation    | la situation       | [see-twah-SYOHN]       |
| question     | la question        | [kess-TYOHN]           |
| attention    | l'attention (f)    | [ah-tahN-SYOHN]        |
| condition    | la condition       | [koN-dee-SYOHN]        |
| solution     | la solution        | [soh-lew-SYOHN]        |
| position     | la position        | [poh-zee-SYOHN]        |
| education    | l'éducation (f)    | [ay-dew-kah-SYOHN]     |
| organisation | l'organisation (f) | [or-gah-nee-zah-SYOHN] |

> 💡 **قانون:** هر کلمه‌ی انگلیسیِ `-tion` دیدی، فرض کن فرانسوی‌اش همونه. تقریباً همیشه درست می‌گی. یه الگو، **صدها**
> کلمه.
