# ⚠️ False Friends (Faux Amis)

## 1. Most Dangerous & Embarrassing 🚨

این‌ها می‌تونن واقعاً شرمنده‌ت کنن. اول این‌ها رو یاد بگیر.

| فرانسوی            | شبیهِ           | معنیِ واقعی                        | اگه منظورت ... بود                                                 |
|--------------------|-----------------|------------------------------------|--------------------------------------------------------------------|
| le préservatif     | "preservative"  | **کاندوم**                         | preservative (ماده‌ی نگهدارنده) → *le conservateur*                |
| blesser            | "to bless"      | **زخمی کردن / آزردن**              | to bless → *bénir*                                                 |
| la déception       | "deception"     | **ناامیدی / سرخوردگی**             | deception (فریب) → *la tromperie*                                  |
| décevoir           | "to deceive"    | **ناامید کردن**                    | to deceive → *tromper*                                             |
| malicieux          | "malicious"     | **شیطون / موذی (بامزه)**           | malicious (بدخواه) → *méchant*                                     |
| la promiscuité     | "promiscuity"   | **شلوغی / ازدحام**                 | promiscuity → *le dévergondage*                                    |
| brutal             | "brutal"        | (اغلب) **ناگهانی**                 | (معنیِ خشن هم داره، ولی اغلب = ناگهانی)                            |
| gros               | "gross"         | **چاق / بزرگ**                     | gross (چندش) → *dégoûtant*                                         |
| la luxure          | "luxury"        | **شهوت / هوس**                     | luxury (تجمل) → *le luxe*                                          |
| l'injure (f)       | "injury"        | **توهین / فحش**                    | injury (جراحت) → *la blessure*                                     |
| le slip            | "slip"          | **شورت (زیرپوش)**                  | slip (لغزش/زیردامنی) → *la combinaison*                            |
| vicieux            | "vicious"       | **فاسد / منحرف**                   | vicious (درنده/بی‌رحم) → *féroce*                                  |
| l'intoxication (f) | "intoxication"  | **مسمومیت (غذایی)**                | intoxication (مستی) → *l'ivresse*                                  |
| **inhabité**       | "inhabited"     | **غیرمسکونی / متروک** (= ضدِّ آن!) | inhabited (مسکونی) → *habité*                                      |
| trivial            | "trivial"       | **زشت / مبتذل** (+ پیش‌پاافتاده)   | (هر دو معنی — به متن دقت کن)                                       |
| excité             | "excited"       | **برانگیخته (جنسی) / هیجان‌زده**   | excited (مشتاق) → *enthousiaste, ravi*                             |
| abuser             | "to abuse"      | **سوءاستفاده / زیاده‌روی**         | to abuse (آزار) → *maltraiter*                                     |
| le bigot           | "bigot"         | **آدمِ متعصبِ مذهبی / مقدس‌مآب**   | bigot (متعصب کور) → *l'intolérant*                                 |
| fat                | "fat"           | **خودبین / مغرور**                 | fat (چاق) → *gros*                                                 |
| le pétrole         | "petrol"        | **نفت خام**                        | petrol (بنزین) → *l'essence*                                       |
| baiser             | "to baise/kiss" | **(فعل) همخوابگی (رکیک!)**         | to kiss → *embrasser* (اما **un baiser** = یک بوسه، بی‌خطر)        |
| débile             | "debilitated"   | **احمق / کودن**                    | debilitated (ناتوان) → *affaibli*                                  |
| intéressé          | "interested"    | **نفع‌طلب / خودخواه**              | interested (علاقه‌مند) → *intéressé par* (با حرف اضافه فرق می‌کند) |
| la confection      | "confection"    | **لباس آماده / دوخت**              | confection (شیرینی) → *la confiserie*                              |
| le pet             | "pet"           | **باد معده (گوز)**                 | pet (حیوان خانگی) → *l'animal de compagnie*                        |
| la bite            | "bite"          | **(رکیک) آلت مردانه**              | bite (گاز) → *la morsure*                                          |

> ⚠️ *inhabité* از همه موذی‌تره: نه‌فقط معنیش فرق داره، بلکه **دقیقاً برعکسِ** چیزیه که فکر می‌کنی. «une île
> inhabitée» = جزیره‌ی **غیرمسکونی**، نه مسکونی.

> ⚠️ *excité* رو با احتیاط استفاده کن: «je suis excité» می‌تونه برداشتِ جنسی بده. برای «هیجان‌زده‌ام» بگو *je suis ravi*
> یا *enthousiaste*.

> ⚠️ *baiser* به‌صورت **اسم** (un baiser) یعنی «یک بوسه» و بی‌خطره، ولی به‌صورت **فعل** معنیِ رکیک داره. برای «بوسیدن»
> همیشه *embrasser*.

> ⚠️ *pet* (باد معده) و *bite* (رکیک) رو بدون که **نگی‌شون** — به‌جای «حیوون خونگی» و «گاز/لقمه» سوتیِ بدی می‌دن.

## 12. Semi-Traps: Sometimes Right 🔍

این‌ها از همه خطرناک‌ترن، چون **گاهی** درست‌ان و **گاهی** غلط — نصف وقت‌ها درست می‌گی و همین تشخیص خطا رو سخت می‌کنه.

| فرانسوی             | شبیهِ           | نکته                                                              |
|---------------------|-----------------|-------------------------------------------------------------------|
| l'expérience (f)    | "experience"    | هم «تجربه» هم **«آزمایش»**                                        |
| l'éducation (f)     | "education"     | اغلب = **«تربیت / ادب»**، نه فقط تحصیل                            |
| la note             | "note"          | هم «یادداشت» هم **«نمره»** هم **«صورت‌حساب»**                     |
| l'occasion (f)      | "occasion"      | هم «مناسبت» هم **«فرصت»** هم **«دستِ‌دوم»** (d'occasion)          |
| la figure           | "figure"        | اغلب = **«صورت / چهره»**                                          |
| la journée          | "journey"       | **«طولِ روز»**، نه «سفر» (سفر → *le voyage*)                      |

> billion» فرانسوی یعنی ۲ تریلیون انگلیسی.

## 13. Gender Changes the Meaning ✨

این کلمات تله‌ی انگلیسی نیستن — تله‌ی **خودِ فرانسوی**‌ان. دلیل دیگه‌ای که چرا باید اسم‌ها رو همراه le/la حفظ کنی: *le
tour* (دور) و *la tour* (برج) دو چیز کاملاً متفاوت‌ان.

| واژه   | با یک جنسیت                 | با جنسیت دیگر              |
|--------|-----------------------------|----------------------------|
| tour   | **le tour** = دور / نوبت    | **la tour** = برج          |
| vase   | **le vase** = گلدان         | **la vase** = لجن / گِل    |
| mode   | **le mode** = روش / حالت    | **la mode** = مد / فشن     |
| livre  | **le livre** = کتاب         | **la livre** = پوند (واحد) |
| poste  | **le poste** = شغل / دستگاه | **la poste** = اداره‌ی پست |
| manche | **le manche** = دسته        | **la manche** = آستین      |
| voile  | **le voile** = حجاب / پرده  | **la voile** = بادبان      |
